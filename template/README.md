# 厉飞雨数学 · LaTeX 专题模板

## 快速使用

1. 复制 `模板_厉飞雨数学.tex` 为新文件，如 `专题02_有理数.tex`
2. 修改封面信息（专题名、年级、要点）
3. 填写知识点和题目内容
4. 编译：`xelatex 专题02_有理数.tex`

## 文件结构

```
template/
├── 模板_厉飞雨数学.tex    ← 空白模板，复制使用
├── 专题01_算术平方根与平方根.tex  ← 已完成的示例
└── README.md              ← 本文件
```

## 可用命令

### 盒子环境

| 命令 | 用途 | 样式 |
|------|------|------|
| `\begin{knowledge}[标题]` | 知识点 | 蓝色标题栏 + 浅蓝背景 |
| `\begin{example}[标题]` | 例题 | 白底 + 淡粉标题栏 |
| `\begin{practice}[标题]` | 跟踪训练 | 白底 + 淡粉标题栏 |
| `\begin{warning}[标题]` | 易错警示 | 浅红背景 + 红色标题 |

### 格式命令

| 命令 | 用途 | 示例 |
|------|------|------|
| `\key{文字}` | 关键词高亮（珊瑚红） | `\key{算术平方根}` |
| `\formula{公式}` | 公式高亮（深蓝） | `\formula{x^2=a}` |
| `\typetag{文字}` | 题型蓝色圆角标签 | `\typetag{题型01}` |

### 排版命令

| 命令 | 用途 |
|------|------|
| `\choices{A}{B}{C}{D}` | 选择题两列排列 |
| `\subqs{(1)}{(2)}{(3)}{(4)}` | 子题 2×2 网格 |
| `\subqsThree{(1)}{(2)}{(3)}` | 子题 2+1 网格 |
| `\subqsFive{(1)}{...}{(5)}` | 子题 2+2+1 网格 |
| `\qitem` | 防跨页（在 enumerate 中使用） |

## 配色方案

```
primary   = RGB(0,100,180)    深蓝（标题、知识点框）
accent    = RGB(220,80,60)    珊瑚红（关键词、易错框）
teal      = RGB(0,140,130)    青色（ subsection 标题）
lightblue = RGB(235,245,255)  浅蓝（知识点背景）
```

## 注意事项

- 编译必须用 `xelatex`（不支持 pdflatex）
- `\typetag` 使用了 `DeclareRobustCommand`，不要改成 `\newcommand`
- 知识点框可跨页，例题/训练框也可跨页
- 封面使用 `titlepage` 环境，自动占满一页
